# Q5. Function: readiness_score


Function Purpose:

The function accepts:

- Attendance percentage
- Diary entries
- Task count


It calculates a readiness score out of 100.

Weightage:

Attendance = 40%

Diary = 20%

Tasks = 40%



Algorithm:

1. Calculate score.

2. Check condition:

IF score >= 85:
    Excellent

ELSE IF score >= 60:
    Good

ELSE:
    Needs Improvement


3. Return final score.
