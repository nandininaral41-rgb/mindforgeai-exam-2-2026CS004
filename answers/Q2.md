# Q2. Data Analytics Pipeline Algorithm

## Input:
Student tabular dataset containing:

- Student ID
- Attendance percentage
- Diary entries
- Tasks completed
- Assignment score
- Lab score


## Process:

1. Read student dataset.

2. Validate data:
   - Check missing values
   - Check incorrect data types

3. Calculate performance signals:
   - Attendance score
   - Task performance
   - Academic performance

4. Calculate final readiness score.

5. Classify students:

IF score >= 85:
    Performance = Excellent

ELSE IF score >= 70:
    Performance = Strong

ELSE IF score >= 50:
    Performance = Developing

ELSE:
    Performance = Support


6. Generate insights:
   - Department average
   - Top performer
   - Students needing support


## Loop Structure:

For each student:
    Read student data
    Calculate score
    Assign performance band


## Edge Cases:

- Missing values
- Empty dataset
- Incorrect input format
- Zero attendance
- Invalid scores


## Output:

Final student analysis containing:
- Readiness score
- Performance band
- Analytics insights
