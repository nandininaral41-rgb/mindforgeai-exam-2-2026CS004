# Q1. Workspace Architecture Map

## Workflow:

VS Code
   ↓
Local Jupyter Notebook
   ↓
Google Colab (Optional)
   ↓
GitHub Repository
   ↓
Google Drive Submission Folder
   ↓
Final Portal Submission


## Tool Responsibilities:

### VS Code:
Used for writing Python code, managing project files, creating folders, and connecting with GitHub.

### Local Jupyter:
Used for creating and executing Python notebooks. It helps in data analysis, testing code, and displaying outputs.

### Google Colab:
An online notebook environment used to run Python code and share notebooks easily.

### GitHub Repository:
Used to store project files such as:
- Notebook (.ipynb)
- Dataset (.csv)
- README file
- Screenshots

### Google Drive Submission Folder:
Used to store:
- Handwritten answer PDF
- Evidence screenshots
- Required submission files

### Final Portal:
The final place where all required links are submitted.
